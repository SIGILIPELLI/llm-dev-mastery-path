# LLM Development Mastery Path

A free, structured, module-wise training program for building applications with
Large Language Models — entry level to master level, with runnable Python code
in every module and a hands-on project at the end of each level. Focused on
applied LLM engineering: prompt engineering, LLM APIs, tool calling, agents,
evaluation, and production platforms.

**Live site:** https://sigilipelli.github.io/llm-dev-mastery-path/

## Contents

- **Level 1 · Entry** — LLM APIs, prompt engineering, structured output, tool calling, a first agent (full lessons)
- **Level 2 · Intermediate** — advanced prompting, prompt caching, multimodal, MCP, evals, multi-agent systems
- **Level 3 · Advanced** — transformer internals, tokenizers, local models (Ollama/vLLM), fine-tuning
- **Level 4 · Master** — training/alignment overview, production LLM platforms, safety & red-teaming, cost at scale

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
